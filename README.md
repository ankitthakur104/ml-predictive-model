# ML Predictive Model

Supervised ML pipeline with Scikit-learn — preprocessing, feature engineering, hyperparameter tuning — achieving 80–85% accuracy.

## Features
- Full preprocessing pipeline (imputation, scaling, encoding)
- Feature engineering and selection
- Model comparison: RandomForest, GradientBoosting, LogisticRegression
- Hyperparameter tuning via GridSearchCV
- Evaluation: accuracy, precision, recall, F1, ROC-AUC
- Experiment tracking with MLflow

## Metrics
- Accuracy: 80–85%
- Evaluated on holdout test set

## Stack
Python · Scikit-learn · Pandas · NumPy · MLflow

## Setup
```bash
pip install -r requirements.txt
python train.py
```
